# Animal Portfolio - Rare Animal Showcase

「最先端の技術と自然の融合」をコンセプトにした、エンジニア向けのポートフォリオサイトです。
スクロールに合わせて珍しい動物たちが顔を出し、クリックすることで詳細な動画紹介がポップアップされます。

## 💡 技術アピールポイント
- **モダンなUI/UX**: グラスモーフィズム（背景ぼかし）とミニマルな白ベースのデザイン。
- **パフォーマンス最適化**: `Intersection Observer API` を使用し、要素が画面内に入った瞬間のみアニメーションを実行。
- **高度なアニメーション**: `GSAP` ライブラリを使用した、滑らかで心地よいインタラクション。
- **保守性の高い設計**: SCSSによる変数管理と、HTML/CSS/JSの分離。

## 🛠 使用技術
- **Frontend**: HTML5, SCSS (Dart Sass), JavaScript (ES6+)
- **Libraries**: [GSAP](https://greensock.com/gsap/) (Animation)
- **Deployment**: GitHub Pages
